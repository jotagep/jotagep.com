# Portfolio de Jotagep

Este repositorio contiene el código fuente del portfolio personal de **Jotagep**.

## Descripción

Es una plantilla de portfolio minimalista construida con **Astro** y **TailwindCSS**, diseñada para ser rápida, accesible y fácil de personalizar.

El sitio web incluye las siguientes secciones:

- **Posts**: Artículos y publicaciones sobre desarrollo y tecnología.
- **Proyectos**: Una muestra de los proyectos realizados.
- **Trabajos**: Historial laboral y experiencia profesional.
- **Contacto**: Formas de contactar con el desarrollador.

## Tecnologías

- [Astro](https://astro.build/)
- [TailwindCSS](https://tailwindcss.com/)
